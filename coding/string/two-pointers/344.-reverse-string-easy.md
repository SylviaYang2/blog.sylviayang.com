# 344. Reverse String (Easy)

````java
```java
public void reverseString(char[] s) {
    int left = 0;
    int right = s.length - 1;

    while (left < right) {
        char temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        left++;
        right--;
    }
}
```
````

**Complexity Analysis**

* Time complexity : $$O(N)$$ to swap $$N/2$$ element.
* Space complexity : $$O(1)$$, it's a constant space solution.
