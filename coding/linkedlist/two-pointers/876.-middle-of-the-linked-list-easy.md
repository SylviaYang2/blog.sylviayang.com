# 876. Middle of the Linked List (Easy)

&#x20;

#### Note：如果链表长度为偶数时，得到的middle node是靠后的那一个。

```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head;
    ListNode fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}
```

### 时间复杂度

**O(N)**，遍历了一遍链表

### 空间复杂度

**O(1)**
